Dataset Requirements Analysis

Dataset title: Air quality. Real-time data

Source: data.europa.eu – Ayuntamiento de Madrid

File format: CSV

License: Open Data License – Ayuntamiento de Madrid

Entities in the selected dataset that can be linked with other datasets:

  • PROVINCIA, MUNICIPIO, and ESTACION identify the monitoring stations and can be linked to geographical datasets or station metadata (e.g., coordinates, district names).
  • ANO, MES, and DIA provide a temporal reference that allows linking with meteorological or traffic datasets collected on the same dates.

Entities in other datasets that can be linked to this dataset:

  • Air quality station metadata datasets (linked via ESTACION or PUNTO_MUESTREO).
  • Meteorological data (linked via ANO, MES, and DIA).
  • Traffic intensity datasets for Madrid (linked via MUNICIPIO and temporal fields).

Requirement Evaluation

Requirement Description Evaluation
R1 Data are in one of the selected smart city domains. Fulfilled. This dataset belongs to the environmental monitoring domain, as it provides real-time measurements of air quality across Madrid, which is a key aspect of Smart City initiatives.
R2 Data are available as a CSV file. Fulfilled. The dataset is provided directly in CSV format and can be downloaded from the official data portal.
R3 Data have an open license so they can be published. Fulfilled. The data is covered by the Open Data License of the Ayuntamiento de Madrid, which explicitly allows reuse, modification, redistribution, and both commercial and non-commercial use, as long as the source is cited and information is not distorted.
R4 Data can be easily linked with generic real-world entities (e.g., locations). Fulfilled. Each observation includes a station identifier and location information. These can be linked to geographic entities such as districts or coordinates of Madrid’s monitoring stations.
R5 (optional) Documentation exists for the data. Partially fulfilled. Basic documentation is provided on the dataset’s webpage, explaining the variables and measurement methodology. However, detailed metadata (e.g., variable units or sensor calibration details) are limited.
R6 (optional) Data can come from multiple data sources. Partially fulfilled. The dataset aggregates data from several monitoring stations across the city (multiple sources in the same network). However, it does not directly integrate with external datasets unless manually linked (e.g., with geographic or meteorological data).
