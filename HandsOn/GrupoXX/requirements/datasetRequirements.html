Functional Requirements

  • BiciMAD dataset
    • R1: The dataset covers bicycle mobility in the city of Madrid (BiciMAD stations).
    • R2: The data are available in CSV.
    • R3: The dataset is licensed under CC BY 4.0 (Attribution 4.0).
    • R4: The resource provides geographic coordinates and location fields that allow linking to real-world entities.
    • R5: Documentation is available on the official portal: BiciMAD stations — PDF documentation.
    • R6: The dataset is listed in other catalogues:
      • datos.gob.es — catalogue entry
      • Google Dataset Search — BiciMAD stations
  • Air Quality dataset
    • R1: The dataset provides real-time air quality measurements in Madrid (pollutant levels at monitoring stations).
    • R2: The data are available in CSV and sometimes JSON via the open data portal.
    • R3: The dataset is licensed under CC BY 4.0 (Attribution 4.0).
    • R4: The dataset includes station identifiers with geographic coordinates, enabling linking to real-world locations.
    • R5: Official documentation exists on the portal: Air quality stations — documentation.
    • R6: The dataset is also referenced in other catalogues:
      • datos.gob.es — catalogue entry
      • Google Dataset Search — Air quality Madrid
  • Traffic Accidents dataset
    • R1: The dataset covers traffic accidents with fatalities in Madrid over the last 10 years.
    • R2: The data are published in CSV format on the open data portal.
    • R3: The dataset is distributed under CC BY 4.0 (Attribution 4.0).
    • R4: Accident records include geographic references (district, coordinates when available) and temporal attributes (year, month, day, hour).
    • R5: Documentation is provided on the portal with data dictionaries and field descriptions: Traffic accidents — documentation.
    • R6: The dataset appears in additional catalogues:
      • datos.gob.es — catalogue entry
      • Google Dataset Search — Traffic accidents Madrid
